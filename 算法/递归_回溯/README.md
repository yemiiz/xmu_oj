# 变形棋盘 · 骑士周游 (Knight's Tour)

## 题目

给定 p 行 q 列的棋盘（p∗q ≤ 26），骑士走标准「日」字，找一条哈密顿路径遍历所有格子。有多条路径时输出字典序最小的。

## 题解算法

**DFS + 回溯 + 字典序排序**

1. **起点按字典序尝试**：A1 → A2 → ... → B1 → B2 → ...
2. **每一步的移动排序**：从当前位置将 8 个可达格子按名字（如 A1, B3）字典序排序，优先尝试最小的
3. 首次找到的完整路径即为全局字典序最小

## 测试

| 棋盘 | 结果 |
|------|------|
| 4×3 | ✅ `A1B3C1A2B4C2A3B1C3A4B2C4` |
| 5×4 | ✅ 有解 |
| 5×5 | ✅ 有解 |
| 4×4 | ❌ 无解 |
| 3×3 | ❌ 无解 |
| 2×N | ❌ 无解 |
